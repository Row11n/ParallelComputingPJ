# 并行分布计算PJ2--PSRS的MPI实现
运行代码前，请先确认安装`mpi4py`
```
pip install mpi4py
```
代码运行指令为
```
mpiexec -n ${num_thread} python main.py -s ${size}
```
其中`num_thread`表示并行线程数，`size`表示数据规模。同时，你可以通过：
```
./run.sh
```
来执行一个写好的脚本，该脚本循环("1000" "5000" "10000" "100000" "1000000")五个数据规模，在每个数据规模下遍历("4" "8" "16" "24" "32")五个规模的并行线程数，并输出结果。
